# 기여하기

기록 제안은 GitHub Issue 또는 Pull Request로 남겨 주세요.

## 기록 템플릿

```md
## 제목

- 날짜:
- 출처:
- 타임스탬프:
- 검증 상태:
- 관련 태그:
- 요약:
```

## 좋은 기여

- 원본 링크가 있는 기록
- 영상의 정확한 타임스탬프가 있는 기록
- 발언을 길게 옮기지 않고 맥락을 요약한 기록
- 밈의 유래와 사용 맥락을 구분한 기록

## 반려될 수 있는 기여

- 출처 없는 소문
- 사생활 침해 가능성이 있는 내용
- 특정 개인이나 집단에 대한 공격
- 투자 매수, 매도, 보유를 권하는 문장
- 저작권이 있는 콘텐츠의 과도한 복제
